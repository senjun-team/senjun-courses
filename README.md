# Курсы Senior Junior

Senior Junior — это курсы от программистов для программистов. Теория без воды и много, мнооого практики. В этом проекте ведется работа над содержимым курсов. А проходить их можно:
- На площадке https://senjun.ru
- В тг-боте https://t.me/sen_jun_bot

Также у нас есть [песочница](https://senjun.ru/playground/) для запуска кода на Rust, Haskell, Go, C++ и Python.

[![Наша площадка и тг-бот](https://github.com/senjun-team/senjun-courses/blob/new-readme/about/senjun_site_and_tg_bot.png)](https://github.com/senjun-team/senjun-courses)

## Наши фишки

Зачем нужны еще одни онлайн-курсы — спросите вы. Наши курсы ориентированы на разработчиков, а не на новичков в мире IT. У нас есть:
- Теория, подкрепленная кодом из реальной жизни. Мы стараемся избегать приевшихся примеров а-ля класс `Animal` или функция `say_hello(name)`.
- Практика, которая подается маленькими порциями. Задачи в online IDE идут прямо по тексту глав. Раз в несколько глав нужно выполнять полноценные проекты.
- Песочница для экспериментов с кодом. В ней по кнопке открываются примеры и задачи из глав.
- Подсказки и полные решения для всех задач.
- Заряженное сообщество, готовое подсказать вам, почему Rust лучше C++ или наоборот. [Присоединяйтесь!](https://t.me/senjun_feedback)

## Обратная связь

Мы рады фидбэку и оперативно на него реагируем! Сообщайте об ошибках и делитесь с нами своим мнением. Для этого есть:
- [Группа в тг.](https://t.me/senjun_feedback)
- Форма обратной связи в конце каждой главы курсов [на сайте.](https://senjun.ru/courses/)
- Команда `/help` в [тг-боте.](https://t.me/sen_jun_bot)
- [Issue](https://github.com/senjun-team/senjun-courses/issues) или [Pull request.](https://github.com/senjun-team/senjun-courses/pulls)
