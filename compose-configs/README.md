# Конфигурационные файлы для Docker Compose

Этот каталог содержит готовые конфигурационные файлы для сервисов docker-compose.

## Как использовать

1. Конфигурационные файлы монтируются в контейнеры через docker-compose.yml
2. Изменения в конфигах применяются после перезапуска контейнера: `docker compose restart <service>`
3. Не требуется пересборка образов для изменения конфигураций

## Безопасность

**Важно**: Конфигурационные файлы могут содержать секреты (пароли, токены, ключи API).
- Не коммитьте реальные credentials в репозиторий
- Для production используйте отдельные конфиги с реальными секретами
